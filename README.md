# FundusData-FS
we originally re-constructed a fundus images dataset, namely FundusData-FS, for few-shot learning recognition task. FundusData-FS consists of 19 different fundus diseases or symptoms and can be found at [[`website`](https://drive.google.com/drive/folders/1zALhjo9nhSsApfteUGL4a0ClQLsovOl3?usp=sharing)]
